# Expense Tracker

A modern **Expense Tracker web application** built using **Flask, SQLite, and Chart.js** that helps users manage and analyze their daily expenses. The application provides a clean dashboard where users can add, edit, delete, and filter expenses while visualizing spending patterns through interactive charts.

---

## Features

- Add new expenses with **description, amount, date, and category**
- Edit existing expense records
- Delete expenses with confirmation
- Filter expenses by **date range and category**
- View **total spending based on selected filters**
- Visualize expense data using **Chart.js charts**
- Export filtered expense data as a **CSV file**
- Clean and responsive UI built with **TailwindCSS**

---

## Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite with SQLAlchemy
- **Frontend:** HTML, TailwindCSS, JavaScript
- **Data Visualization:** Chart.js
- **Templating Engine:** Jinja2
